# Determinacion-de-mayor-entero-estricto-de-3-numeros
El siguiente programa determinara el mayor entero estricto de entre 3 nÃºmeros sin el uso de operadores logicos

A=int(input("Introduce el primer numero:"))

B=int(input("Introduce el segundo numero:"))

C=int(input("Introduce el segundo numero:"))

D=0
if A>0:
	D=1
E=0
